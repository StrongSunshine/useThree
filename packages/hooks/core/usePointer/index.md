# 交互操作

事件监听交互对应的3Dobeject

## Usage

```ts
const pointer = usePointer({
    camera: camera
    domElement: domElement
    intersectObjects: intersectObjects
})
```

### `camera`

摄像机实例

- 类型：Three.camera

### `domElement`

- 类型：Three.WebGLRenderer.domElement

### `intersectObjects`

- 类型：Array
